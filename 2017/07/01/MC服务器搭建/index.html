MC服务器搭建

2017-07-01 PV:

1.购买服务器(已有的请忽略)

我购买的是腾讯云

接下来就请自行查阅百度——putty,ftp软件 以便后面的使用
熟悉自己的ip和root账号密码.

2.接下里就重点了(个人总结的mc服务器地点)

mc服务器下载地点:

https://mcversions.net/


https://files.minecraftforge.net/

第一个网站是原生的，请自行选择自己的版本

第二个网站是forge版，请自行选择自己的版本。—ps:forge的安装后含原生版本(23333…)

3.在本地配置mc-server

打开下载好的服务器文件

若是forge版的，请选择install server

无论如何设置路径都是红色的哦，所以不需要管…

那么，接下来,它会自动生成一堆文件

找到 eula.txt文件，然后把其中的false改为true

重新打开服务器文件,一件神奇的事情发生了,又多了一些文件

找到 server.properties文件,把online-mode=true改为flase，这样盗版玩家就可以乱入了

再次打开服务器文件,正常运行p(^_^)q！

4.上传本地服务器

登陆ftp进入根目录

在/home 中上传你的mc文件(需要用到ftp软件—sftp登陆,账号就是root，密码就是root的密码)

putty中输入

配置+运行

5.Have Fun!

累死了,就写了一点点,写不齐全(懒癌QAQ…),可能错误多，没办法.
